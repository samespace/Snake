# Snake

Nokia-style Snake game built with React + TypeScript + Vite.

## Play

```bash
npm install
npm run dev
```

## Controls

| Key | Action |
|-----|--------|
| Arrow keys / WASD | Move |
| Enter / Space | Start · Pause · Resume · Restart |
| Escape | Pause / Resume |
| Swipe | Mobile control |

## Modes

| Mode | Speed |
|------|-------|
| Easy | 200ms |
| Medium | 140ms |
| Hard | 90ms |

Speed increases as you eat food. High score saved locally.
